# Amiineecmoii — GitHub Pages Website

This is a static hacker-style portfolio website for **Amine el Jabri (Amiineecmoii)**.

## Deploy on GitHub Pages (username.github.io)

1. Create a repository named exactly: `YOURUSERNAME.github.io`
2. Upload everything in this folder to the repo root
3. Go to **Settings → Pages**
4. Set:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
5. Your site will be live at: `https://YOURUSERNAME.github.io`

## Customize

Open `index.html` and edit:
- Social links in the **Contact** section
- Skills / Featured / Writeups text

Design files:
- `assets/css/style.css`
- `assets/js/script.js`
